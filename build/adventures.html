Projects

LED Foot

Hardware and software for RGB LEDs to improve quality-of-life during the winter months by providing atmospheric and reactive lighting effects to a living space.

Volumetric Lighting

Artist-driven volumetric lighting for a custom 3D in-browser game engine written in Rust.

Ray Tracer

CPU Ray Tracer written in Rust. Also compiled to WebAssembly and available on GitHub.

Real-Time Sound Propagation

Real-time engine to simulate propagation of sound throughout virtual spaces, written in C++.

Scriptable Game

Scriptable 3D game engine for browsers, written in Rust and JavaScript.

2D Game Framework

A simple 2D component-based game framework for browsers, written on top of p5.js.

Deferred Rendering Engine

A second attempt at a graphics engine in Rust, including a comparison of forward, deferred, and deferred + light volumes rendering techniques.

Maze Game

Maze game with raw OpenGL, including a first attempt at a graphics framework in Rust.

Multi-Agent Motion Planning

3D multi-agent motion planning, including A* search, a probabilistic road map, and boids algorithm, written in C++.

Cloth Simulation

Cloth simulation, including rigid bodies, written in C++

Particle System
